TROUBLESHOOTING QUICK REFERENCE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Problem: Fonts not loading
Solution: Check Google Fonts URL in styles.css line 1
          Clear browser cache (Ctrl+Shift+Delete)
          Check network tab in DevTools

Problem: Colors look different
Solution: Check hex codes match specification
          → #d4af37 (gold), #0a0a0a (black), #f5f5f5 (text)
          Verify CSS variables in @theme section

Problem: Hover animations stuttering
Solution: Check DevTools Performance tab
          Reduce animation complexity
          Enable hardware acceleration
          Use will-change hints

Problem: Layout broken on mobile
Solution: Check responsive breakpoints (sm:, md:, lg:)
          Test on actual mobile device
          Check viewport meta tag

Problem: Accessibility score low
Solution: Use Lighthouse audit
          Fix contrast ratio issues
          Improve focus states
          Add ARIA labels where needed
