## 🎨 Design Specifications

### Color Palette
```css
Gold:          #d4af37 (Pantone 871 C)
Black:         #0a0a0a (pure black)
Paper:         #1a1a1a (dark gray)
Text:          #f5f5f5 (bright white)
Accent:        #8b9dc3 (blue-gray)
```

### Font Stack
```css
Display: "Playfair Display" (headings)
Serif:   "Crimson Text" (body)
UI:      "Inter" (navigation/buttons)
```

### Easing Curves
```css
silk:       cubic-bezier(0.22, 1, 0.36, 1)
editorial:  cubic-bezier(0.16, 1, 0.3, 1)
premium:    cubic-bezier(0.2, 0, 0.38, 0.9)
pop:        cubic-bezier(0.34, 1.56, 0.64, 1)
```

### Transition Timing
- Interactive elements: **400ms**
- Large changes: **500ms**
- Emphasis elements: **700ms**
